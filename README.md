# Jadoo Landing Page
## 🌐 Live Demo
[View Jadoo Website](https://amazing-florentine-b44b05.netlify.app/)

##  Overview
Jadoo is a responsive **travel and tourism landing page** developed during my training period.  
The project was designed to practice advanced UI/UX concepts with **Bootstrap**, implement a **Dark/Light Mode toggle**, and experiment with more complex design structures to simulate real-world scenarios.

##  Tech Stack
- HTML  
- CSS  
- Bootstrap  
- JavaScript  

##  Features
- Fully responsive design across mobile, tablet, and desktop.  
- **Dark/Light Mode** functionality for modern user experience.  
- Clean travel-themed layout with sections for destinations, services, and offers.  
- More complex design elements compared to basic landing pages, reflecting professional practice.  
- Organized structure: Header, Hero, Destinations, Services, Testimonials, Footer.  

##  Outcomes
- Learned how to integrate **Dark/Light Mode** into a Bootstrap-based project.  
- Improved skills in responsive design and layout complexity.  
- Strengthened understanding of real-world landing page requirements in the travel industry.  
- Built confidence in handling more advanced UI/UX challenges during training.
